## Contributing

### Installation

```bash
# Clone the project
git clone git@github.com:studiometa/vue-mapbox-gl.git
# Cd in the repository
cd vue-mapbox-gl
# Install dependencies
yarn
```

### Useful commands

```bash
# Build for deployment
yarn run build
# Test before deployment
yarn run test
# Lint files
yarn run lint
# Fix linting errors
yarn run fix
```
